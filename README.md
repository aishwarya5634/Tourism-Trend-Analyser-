# Tourism Trend Analyser 
Data Analytics Project
•	Solved the business problem of identifying which states, seasons, and segments (domestic vs foreign, purpose of visit) drive the highest and most sustainable tourism revenue and growth.
•	Web Scraping and data extraction of the real-time data on tourism records across all states of India from Govt websites.
•	Data Cleaning – Validated data types, Standardised dates and checking for missing values
•	Feature Engineering – adding 10+ new features like revenue split, seasonality index and ratio, etc.
•	Advanced Analytics and Insights, data Exploration, Aggregation. Key metrics and KPIs
